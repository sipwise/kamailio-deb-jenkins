Welcome to the Kamailio Debian repositories

This is the Debian/Ubuntu package repository of Kamailio, powered by jenkins-debian-glue.org and sponsored by Sipwise.

Build information available at Jenkins system.

RTPengine builds

Usage instructions:

    % wget -O- https://deb.kamailio.org/kamailiodebkey.gpg | gpg --dearmor | sudo tee /usr/share/keyrings/kamailio.gpg

add key info in repository definition:

    deb [signed-by=/usr/share/keyrings/kamailio.gpg] http://kamailio.sipwise.com/debian/kamailio${VER} ${CODENAME} main

Old method:

    % wget -O- http://kamailio.sipwise.com/debian/kamailiodebkey.gpg | sudo apt-key add -
