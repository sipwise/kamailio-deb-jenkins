Welcome to the RTPengine Debian repositories

This is the Debian/Ubuntu package repository of RTPengine, powered by jenkins-debian-glue.org and sponsored by Sipwise.

Build information available at Jenkins system.

Usage instructions:

% wget -O- http://kamailio.sipwise.com/debian/kamailiodebkey.gpg | sudo apt-key add -
